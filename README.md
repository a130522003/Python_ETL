# Python_ETL

# CSV 批次上傳到 MySQL 的 Python ETL 工具

這是一個使用 Python 開發的 **ETL 工具**，可以將本地 CSV 檔案自動上傳到 MySQL 資料庫，具備資料清理、例外處理、日誌紀錄與環境設定功能，適合作為數據分析與自動化流程的一部分。

---

## 功能特色

- **CSV 讀取與資料清理**  
  - 自動清理欄位名稱，移除空格與括號  
  - 支援多種 CSV 格式  

- **資料庫自動化上傳**  
  - 連接 MySQL，將資料寫入指定資料表  
  - 若資料表不存在，自動建立新表  
  - 若資料表已存在，自動以 `append` 模式新增資料，避免覆蓋舊資料  

- **ETL 自動化與安全性**  
  - 使用 `.env` 檔案管理資料庫帳號密碼  
  - 支援資料庫連線池，提高效率  
  - 節省手動操作時間，方便每日或定期批次上傳  

- **例外處理與錯誤追蹤**  
  - 使用 `try/except` 捕捉錯誤，避免程式中斷  
  - 使用 `logging` 記錄程式執行狀態與錯誤資訊到 `csv_to_sql.log`
